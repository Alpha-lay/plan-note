# Roadmap

This roadmap explains the next practical milestones for PlanNote. The focus is to keep the project useful as a small personal planning assistant while adding AI features that are easy to understand, test, and maintain.

## Current Priorities

- Improve README, setup instructions, and contributor documentation
- Remove committed secrets and document safe reminder key handling
- Add a clear AI feature plan for OpenAI API credit usage
- Add basic quality checks before larger feature work

## AI-Assisted Planning

Planned OpenAI-powered features:

- Task breakdown: turn a large task into smaller steps
- Daily planning: suggest a simple plan from tasks and notes
- Note summarization: summarize notes into useful planning context
- Priority suggestions: recommend task priority from due date, time, and note content
- Reminder drafting: generate clear reminder titles and descriptions

## Security Improvements

- Keep SendKey and future API keys out of the repository
- Add `.env.example` if backend or API configuration is introduced
- Review notification code for unsafe secret handling
- Document local data storage behavior clearly
- Add dependency and build checks

## Community Improvements

- Add screenshots or a short demo GIF
- Publish a hosted demo when the app is ready
- Create beginner-friendly issues
- Add clear issue templates for bugs and feature requests
- Share implementation notes for the AI features

## Possible Future Work

- Export and import local planning data
- Better mobile layout polish
- Weekly review view
- Optional encrypted local backup
- Multi-language interface support
